# Adding Subattributes

> **Note:** Subattributes allow you to add more detailed information to your features, such as inclination, condition, or other properties.

## Add New Subattribute
To add a new subattribute for a feature (e.g., utility pole), use the "Add new subattribute" button.

![Add new subattribute button](/images/image42.png)

## Configure Subattribute
In the pop-up window, indicate the name of the subattribute, select the data type, and select a data source.

![Subattribute configuration popup](/images/image38.png)

## Example: Inclination
For example, to identify the inclination subattribute of a utility pole:
- Select the data type (e.g., integer)
- Associate the data source (e.g., "inclination")

![Inclination subattribute example](/images/image35.png)

## Restrict Subattribute Values
You may want to restrict subattribute values. For example, create a list representing the conditions.

![Restrict subattribute values](/images/image56.png)
